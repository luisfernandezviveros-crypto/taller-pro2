# Programación en Java

## 15. ¿Cuáles son los tipos de datos primitivos en Java?
Los tipos de datos primitivos en Java se utilizan para almacenar información simple y
directa, como valores numéricos, caracteres individuales y datos lógicos. Por ejemplo,
`int` se usa para números enteros, `double` para valores con decimales, `char` para letras
y `boolean` para valores de verdadero o falso.

---

## 16. ¿Cómo funcionan las estructuras de control de flujo en Java?
Las estructuras de control permiten definir el camino que seguirá un programa durante
su ejecución. Mediante `if` y `else` se toman decisiones, con `switch` se selecciona una
opción entre varias posibles y con los bucles se repiten instrucciones de forma controlada.

---

## 17. ¿Por qué es importante usar nombres significativos?
Utilizar nombres descriptivos en variables y métodos facilita la comprensión del código.
Esto permite que el programa sea más claro, más fácil de modificar y más sencillo de
mantener a largo plazo.

---

## 18. ¿Qué es la Programación Orientada a Objetos?
La Programación Orientada a Objetos es un enfoque de desarrollo que organiza el software
usando objetos. Cada objeto representa una entidad y contiene información y acciones que
definen su comportamiento.

---

## 19. ¿Cuáles son los cuatro pilares de la Programación Orientada a Objetos?
La Programación Orientada a Objetos se basa en cuatro principios fundamentales:
- Encapsulamiento
- Herencia
- Polimorfismo
- Abstracción  

Estos principios permiten crear programas más organizados, reutilizables y fáciles de
entender.

---

## 20. ¿Qué es la herencia y cómo se usa en Java?
La herencia es un mecanismo que permite a una clase adquirir atributos y métodos de otra
clase. En Java se implementa mediante la palabra clave `extends`.

---

## 21. ¿Qué son los modificadores de acceso en Java?
Los modificadores de acceso determinan el nivel de visibilidad de clases, métodos y
atributos. Los más utilizados en Java son `public`, `private` y `protected`.

---

## 22. ¿Qué es una variable de entorno y por qué es importante?
Una variable de entorno es un valor definido en el sistema operativo que puede ser usado
por los programas. Son importantes porque permiten configurar aplicaciones sin necesidad
de modificar el código fuente.
